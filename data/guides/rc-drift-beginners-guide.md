## ğŸ¯ **4. Learning Basic Drift Techniques**

### **Step 1: Donut Drifting**
**Practice this first!**
- Find smooth surface (basketball court, garage)
- Start with small circles
- Focus on throttle control
- Practice both directions

**Key Skills:**
- Throttle modulation
- Steering correction
- Gyro sensitivity adjustment

### **Step 2: Figure-8 Drifting**
**Master transitions between directions**
- Start with large figure-8 patterns
- Focus on smooth direction changes
- Practice weight transfer timing

### **Step 3: Track Drifting**
**Apply skills to course layouts**
- Follow simple track layouts
- Practice entry/exit speeds
- Learn to link corners

## ğŸ”‹ **5. Battery and Charging Basics**

### **Battery Selection**
- **Type:** 2S LiPo (7.4V)
- **Capacity:** 3000-5000mAh
- **C-Rating:** 25-35C (beginner)
- **Connector:** XT60 (most common)

### **Safe Charging Practices**
- **Always use balance charger**
- **Charge at 1C rate** (3A for 3000mAh)
- **Use fireproof charging bag**
- **Never leave charging unattended**

### **Storage Guidelines**
- **Store at 3.8V per cell**
- **Use storage charge function**
- **Keep in cool, dry place**
- **Check voltage monthly**
