## üõû **2. Drift Tire Upgrades ‚Äì The Most Important Mod**

![Drift tire comparison showing different compounds and sizes](/static/images/articles/rc-drift-upgrades-mods-guide-image-2.webp)
*Different drift tire compounds and sizes for various surfaces ‚Äì choose the right one for your track conditions.*

### **Tire Compound Selection**

| Surface | Recommended Compound | Characteristics |
|---------|---------------------|-----------------|
| **Concrete** | Hard Plastic | Maximum slide, low grip |
| **Wood Floor** | Medium Hard | Balanced slide/grip |
| **Tile/Smooth** | Soft Plastic | More grip, controlled slides |
| **Carpet** | Very Soft | Maximum grip for low-traction surfaces |

### **Tire Width Considerations**
- **Narrow tires:** Faster steering response, easier to initiate drifts
- **Wide tires:** More stability, better for high-speed drifts
- **Standard:** 26-28mm width for 1/10 scale cars

### **Installation Tips**
- Clean wheels thoroughly before mounting
- Use tire glue for permanent installation
- Balance tires if using foam inserts
- Check for consistent contact patch

## üéÆ **3. Gyro Stabilizer Upgrades ‚Äì Precision Control**

![High-performance gyro stabilizer installation and wiring diagram](/static/images/articles/rc-drift-upgrades-mods-guide-image-3.webp)
*Professional gyro stabilizer installation showing proper wiring and mounting position for optimal performance.*

### **Gyro Types and Features**

| Type | Features | Best For |
|------|----------|----------|
| **Basic 2-Channel** | Simple gain control | Beginners |
| **Advanced Digital** | Multiple modes, tuning | Intermediate |
| **Competition Grade** | Real-time adjustment | Experts |

### **Installation Guidelines**
```
1.  Mount gyro as close to chassis center as possible
2.  Ensure gyro is level with chassis
3.  Use double-sided tape or proper mounts
4.  Route wires neatly to avoid interference
5.  Set initial gain at 50% and adjust
```

### **Tuning Parameters**
- **Gain:** Controls sensitivity (start at 50%)
- **Mode:** Normal/AVC/Angle Hold
- **Response Speed:** How quickly gyro reacts
- **Endpoint Adjustment:** Limits maximum correction
